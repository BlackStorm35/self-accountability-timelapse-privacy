# self-accountability-timelapse-privacy
Last updated: [2026-03-03]

Summary
Self Accountability Timelapse helps you create an MP4 timelapse of your own study/work sessions. It only captures screenshots when you start a session, and only on sites you explicitly allowlist.

Data the extension processes
	1.	Website content (screenshots / “shots”)

	•	When you start a session, the extension may capture periodic screenshots of the active tab on allowlisted sites.
	•	These screenshots are uploaded to cloud storage to generate your MP4 timelapse.

	2.	Extension identifiers & diagnostics

	•	The extension generates/uses an Install ID and may send basic app diagnostics needed to operate the service (for example: extension version, selected plan/interval, timezone, session IDs, and quota counters).

	3.	Local-only data (stored in your browser)

	•	Settings like your allowlisted domains, plan selection, UI preferences, and session summaries are stored locally using browser storage to keep the extension working consistently.

How data is used (single purpose)
Data is used only to:
	•	capture frames you requested during a running session,
	•	upload frames and generate the MP4 timelapse,
	•	enforce quotas/rate limits to keep the service sustainable, and
	•	troubleshoot issues when you contact support.

Data retention
	•	Uploaded frames are stored temporarily to build your MP4, then deleted automatically based on the current retention policy shown in the extension UI (for example, 1 day on the free plan).
	•	Local settings and local session summaries remain in your browser until you remove the extension or clear extension storage.

Data sharing / transfers
	•	Your data is not sold.
	•	Data is transferred only to service providers required to deliver the timelapse feature (for example, cloud storage/processing used to upload frames and build the MP4).
	•	Sharing the final MP4 is optional and manual (you choose if/where to share it).

Advertising
No screenshots or other collected data are used for advertising, profiling, or interest-based targeting.

User control
	•	You control capture with Start / Pause / Stop.
	•	The extension does not capture when a session is stopped, and it will not capture outside your allowlist.
	•	You can remove allowlisted permissions at any time from the extension settings or your browser site permissions.

Security
Reasonable technical measures are used to protect data in transit and at rest. (No system is perfect)
